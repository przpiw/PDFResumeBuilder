# PDF Resume Builder
[Live on](https://pdfr-esume-builder.vercel.app/)
## Editor & PDF Viewer allow to customize resume template
- Built with react & PDF Renderer
- Utilizes Styles, Images, SVGs
- Highly customizable template
- Clean Design
